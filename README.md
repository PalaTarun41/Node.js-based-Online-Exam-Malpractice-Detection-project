
# Online Exam Malpractice Detection (Rule-Based)

## Problem
Online exams are vulnerable to cheating via tab switching, copying, and multiple logins.

## Solution
A rule-based system that detects suspicious activities using browser events.

## Features
- Tab switch detection
- Copy-paste blocking
- Suspicion scoring
- Admin monitoring

## Tech Stack
HTML, CSS, JavaScript, Node.js, MySQL

## How to Run
1. npm install express
2. node backend/server.js
